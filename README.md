# Customer Churn Prediction

This project predicts whether a customer will churn using machine learning techniques.

## 🔧 Techniques Used
- Data Cleaning & Preprocessing
- Feature Encoding
- Random Forest Classifier

## 📊 Result
Achieved ~78% accuracy in predicting customer churn.

## 🔍 Key Insights
- Customers with higher charges are more likely to churn
- Customers with shorter tenure tend to leave early
- Certain services and payment methods influence churn behavior

## 📁 Dataset
Telco Customer Churn Dataset (Kaggle)

## 💡 Conclusion
This project demonstrates how machine learning can help businesses identify at-risk customers and improve retention strategies.
